Products.

Donation Money iOS Apps

DoNe adalah aplikasi yang membantu masyarakat untuk mendonasikan uang atau barangnya kepada yang berkebutuhan. Tujuan adanya aplikasi ini adalah membangkitkan kesadaran masyarakat untuk berdonasi dengan menggunakan Twibbon dan Sosial Media sebagai platformnya.

Aplikasi ini saya buat pada saat saya mengikuti pelatihan di iOS Foundation pada Agustus sampai September 2019. Disini saya menggunakan aplikasi Sketch untuk membuat Prototyping dan menggunakan metode Human Interface Guidelines dari Apple Developer.

The Live Longer Hospital Apps

Live Longer Hospital Apps merupakan aplikasi yang saya buat untuk Ujian Akhir Semester. Aplikasi ini bertujuan untuk membantu masyarakat untuk melakukan booking menggunakan aplikasi untuk berobat pada dokter, sehingga pasien tidak perlu mengantri lagi.

Aplikasi ini saya desain menggunakan Adobe XD dan UI Aplikasi ini saya menggunakan platform iOS
